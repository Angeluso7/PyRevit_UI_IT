Introducción

Propósito de esta guía

Esta guía está diseñada para usuarios que utilizan las herramientas de la Extensión PyRevit Infotécnica en su trabajo diario, sin necesidad de conocimientos de programación.

Aprenderás:

·         Qué hace cada botón y para qué sirve

·         Cuándo usar cada herramienta

·         Cómo usar cada función paso a paso

Qué es la Extensión PyRevit Infotécnica

Es un conjunto de herramientas que aparecen como una pestaña adicional en Revit, diseñadas específicamente para proyectos de transmisión eléctrica de Infotécnica.

Estas herramientas te ayudan a:

·         Gestionar datos del proyecto de forma rápida

·         Asignar y revisar códigos internos (CodIntBIM)

·         Editar información de muchos elementos a la vez

·         Exportar planillas en el formato que requiere el Coordinado Eléctrico Nacional

Requisitos para usar la extensión

Antes de comenzar, asegúrate de tener:

·         Revit instalado (versiones compatibles con pyRevit)

·         PyRevit instalado en tu equipo

·         Acceso al archivo host plantilla de Infotécnica (ya configurado con parámetros y tablas)

·         Permisos para editar el modelo

Estructura General de la Interfaz

Dónde encontrar las herramientas

Cuando abres Revit con la extensión instalada, verás una pestaña nueva en la cinta superior con el nombre de Infotécnica.

Dentro de esta pestaña hay cuatro paneles principales:

Panel

Para qué sirve

menu

Gestionar datos del proyecto, cargar información y editar parámetros

revision

Ver y revisar elementos según su estado, filtrar y validar información

exportacion de Planillas

Crear archivos Excel de planillas de valorización

documentacion

Acceder a ayuda y documentos de consulta

 

Cómo identificar los botones

·         Botones simples: Haces clic y ejecutan una acción

·         Desplegables: Al hacer clic se abre un menú con más botones relacionados

·         Iconos: Cada botón tiene un dibujo que representa su función

·         Texto de ayuda: Si pasas el mouse sobre un botón (sin hacer clic), aparece una descripción breve

Panel 01: "menu"

Este panel contiene todas las herramientas para gestionar información del proyecto y editar datos de elementos.

Desplegable "proyecto"

Contiene dos botones para registrar datos generales del proyecto según el tipo de cliente.

Botón "coordinador"

Qué hace:
Abre una ventana donde puedes ingresar o editar los datos del proyecto cuando trabajas para el Coordinador Eléctrico Nacional.

Cuándo usarlo:
Al inicio del proyecto o cuando necesites actualizar información como el nombre del proyecto, NUP, región, fecha de puesta en servicio, etc.

Cómo usarlo:

1.       Haz clic en el botón "coordinador"

2.      Se abre la ventana "Gestor de Proyectos"

3.      Puedes:

o    Crear un proyecto nuevo escribiendo el nombre

o    Seleccionar un proyecto existente de la lista

o    Hacer clic en "Editar" para cambiar los datos (NUP, propietario, tensión, decreto, etc.)

o    Hacer clic en "Eliminar" si quieres borrar un proyecto de la lista

4.      Cuando termines, haz clic en "Aceptar" para guardar los cambios

Ejemplo práctico:
Si empiezas a trabajar en un nuevo proyecto para el Coordinador, usas este botón para registrar toda la información oficial (NUP, nombre, región, tensión) que después aparecerá en los reportes.

Botón "saesa"

Qué hace:
Abre una ventana donde puedes ingresar o editar los datos del proyecto cuando trabajas para Grupo SAESA.

Cuándo usarlo:
Al inicio del proyecto o cuando necesites actualizar información específica de SAESA (código de proyecto, tipo de instalación, fechas de operación, estado, etc.).

Cómo usarlo:

1.       Haz clic en el botón "saesa"

2.      Se abre la ventana "Datos del Proyecto"

3.      Si ya hay datos registrados, aparecen en los campos

4.      Haz clic en "Editar" para poder modificar la información

5.       Completa o modifica los campos:

o    Proyecto (código)

o    Tipo de instalación

o    Fechas de entrada/salida de operación

o    Propietario

o    Estado

o    Comuna

o    Descripción

6.      Haz clic en "Aceptar" para guardar

Ejemplo práctico:
Si el proyecto es una subestación para SAESA, usas este botón para registrar que es tipo "Subestaciones", la comuna "EL BOSQUE", el propietario "SAESA" y el estado "EN OPERACIÓN".

Desplegable "datos"

Contiene botones para sincronizar información entre el modelo y archivos externos (como Excel).

Botón "Actualizacion de datos"

Qué hace:
Actualiza la información del modelo usando datos nuevos que vienen de archivos Excel o bases de datos externas.

Cuándo usarlo:
Cuando el equipo de ingeniería actualiza información técnica (tensiones, estados, propietarios, etc.) en un Excel centralizado y necesitas reflejar esos cambios en el modelo.

Cómo usarlo:

1.       Asegúrate de que el archivo Excel actualizado esté en la ubicación correcta (configurada previamente)

2.      Haz clic en "Actualizacion de datos"

3.      La herramienta lee el archivo y compara con lo que hay en el modelo

4.      Actualiza automáticamente los elementos que cambiaron

5.       Al terminar, muestra un resumen:

o    Cuántos elementos se actualizaron

o    Qué parámetros cambiaron

o    Si hubo errores

Ejemplo práctico:
Cada lunes, ingeniería actualiza un Excel con cambios de tensiones y estados. Tú ejecutas "Actualizacion de datos" y en segundos el modelo refleja todos los cambios sin editar manualmente cada elemento.

Botón "Carga de datos"

Qué hace:
Importa información desde un archivo Excel y la carga por primera vez en los elementos del modelo.

Cuándo usarlo:
Al inicio del proyecto, después de asignar códigos internos (CodIntBIM) a los elementos, cuando recibes un Excel con toda la información técnica inicial que debe entrar al modelo.

Cómo usarlo:

1.       Asegúrate de que el Excel tenga una columna llamada "inf-id" que coincida con los códigos de tus elementos

2.      Haz clic en "Carga de datos"

3.      Selecciona el archivo Excel

4.      La herramienta lee el Excel y busca cada elemento por su inf-id

5.       Actualiza los parámetros de cada elemento con los valores del Excel

6.      Al terminar, muestra un resumen de cuántos elementos se actualizaron

Ejemplo práctico:
Recibes un Excel con 500 líneas de datos técnicos. En lugar de copiar y pegar manualmente en Revit, usas "Carga de datos", seleccionas el Excel y en minutos los 500 elementos tienen toda su información cargada.

Desplegable "editar"

Contiene botones para modificar parámetros de elementos, ya sea uno por uno o en grupos.

Botón "Parametros por elementos"

Qué hace:
Te permite seleccionar un elemento individual en el modelo y editar sus parámetros en una ventana dedicada.

Cuándo usarlo:
Cuando necesitas corregir o actualizar información de un elemento específico sin afectar otros.

Cómo usarlo:

1.       Haz clic en "Parametros por elementos"

2.      Selecciona un elemento en la vista de Revit (por ejemplo, una torre, un equipo, una línea)

3.      Se abre una ventana mostrando:

o    Información básica del elemento (categoría, familia, código)

o    Lista de todos sus parámetros con valores actuales

4.      Modifica los valores que necesites cambiar

5.       Haz clic en "Aceptar" para aplicar los cambios

6.      Si quieres, puedes seleccionar otro elemento y seguir editando

Ejemplo práctico:
Detectas que una torre tiene mal escrito el "Propietario". Usas este botón, seleccionas esa torre, corriges el valor y guardas. Solo esa torre se modifica.

Botón "Parametros por planilla"

Qué hace:
Abre una tabla (similar a Excel) donde puedes editar parámetros de muchos elementos a la vez, agrupados por planilla.

Cuándo usarlo:
Cuando necesitas cambiar el mismo parámetro en muchos elementos simultáneamente (por ejemplo, cambiar el "Estado" de 50 líneas de "Proyectado" a "Existente").

Cómo usarlo:

1.       Haz clic en "Parametros por planilla"

2.      Selecciona la planilla con la que quieres trabajar (por ejemplo, "CM01 - Líneas de Transmisión")

3.      Se abre una tabla con:

o    Cada fila es un elemento

o    Cada columna es un parámetro

4.      Puedes:

o    Hacer clic en una celda y cambiar el valor

o    Seleccionar varias filas y cambiarlas todas a la vez

o    Filtrar elementos por valores específicos

5.       Cuando termines, haz clic en "Aceptar" para guardar todos los cambios

Ejemplo práctico:
Necesitas cambiar el "Propietario" de 30 subestaciones. Abres la planilla CM02, seleccionas las 30 filas, cambias el valor en la columna "Propietario" y guardas. Las 30 subestaciones se actualizan al mismo tiempo.

Panel 02: "revision"

Este panel contiene herramientas para visualizar, filtrar y validar el estado de los elementos del modelo.

Desplegable "filtrado y seleccion"

Botones para mostrar solo ciertos elementos según diferentes criterios.

Botón "Por Codigo Interno"

Qué hace:
Filtra la vista para mostrar solo los elementos que tienen un código interno (CodIntBIM) específico.

Cuándo usarlo:
Cuando quieres revisar visualmente todos los elementos que comparten el mismo código.

Cómo usarlo:

1.       Haz clic en "Por Codigo Interno"

2.      Aparece una lista con todos los códigos que existen en el modelo

3.      Selecciona el código que quieres revisar (por ejemplo, "CM01-00125")

4.      La vista muestra solo los elementos con ese código

5.       Los demás elementos se ocultan o atenúan

6.      Para volver a ver todo, usa el botón "Reestablecer vista"

Ejemplo práctico:
Quieres verificar que todos los elementos con código CM03-00045 están en el lugar correcto. Usas este botón, seleccionas ese código y la vista te muestra solo esos elementos para revisarlos fácilmente.

Botón "Por Elemento"

Qué hace:
Aísla un elemento específico que seleccionas, ocultando todo lo demás para que puedas inspeccionarlo en detalle.

Cuándo usarlo:
Cuando necesitas concentrarte en un elemento individual sin distracciones visuales.

Cómo usarlo:

1.       Haz clic en "Por Elemento"

2.      Selecciona el elemento que quieres ver en la vista de Revit

3.      Solo ese elemento queda visible

4.      Puedes inspeccionarlo con tranquilidad

5.       Para volver a ver todo, usa "Reestablecer vista"

Ejemplo práctico:
Necesitas verificar las conexiones de un equipo específico. Lo aíslas con este botón y puedes verlo claramente sin otros elementos que estorben la vista.

Botón "Por Planilla IT"

Qué hace:
Muestra solo los elementos que pertenecen a una planilla específica que seleccionas.

Cuándo usarlo:
Cuando quieres revisar visualmente todos los elementos de una categoría (por ejemplo, todas las subestaciones o todas las líneas).

Cómo usarlo:

1.       Haz clic en "Por Planilla IT"

2.      Aparece una lista con todas las planillas del modelo

3.      Selecciona la planilla que quieres ver (por ejemplo, "CM02 - Subestaciones")

4.      La vista muestra solo los elementos de esa planilla

5.       Para volver a ver todo, usa "Reestablecer vista"

Ejemplo práctico:
Quieres verificar que todas las subestaciones están correctamente ubicadas. Seleccionas la planilla CM02 y la vista te muestra solo las subestaciones, facilitando la revisión.

Botón "Reestablecer vista"

Qué hace:
Restaura la vista para que vuelvas a ver todos los elementos como antes de aplicar filtros.

Cuándo usarlo:
Después de usar cualquiera de los botones de filtrado anteriores, cuando quieras volver a la vista completa.

Cómo usarlo:

1.       Haz clic en "Reestablecer vista"

2.      Todos los elementos vuelven a ser visibles

3.      La vista vuelve a su estado normal

Ejemplo práctico:
Terminaste de revisar elementos filtrados y quieres seguir trabajando con el modelo completo. Un clic en este botón y listo.

Desplegable "herramientas"

Contiene botones para validar y exportar información del modelo.

Botón "Planillas vs Modelo"

Qué hace:
Compara los elementos que aparecen en las planillas con los elementos reales del modelo, identificando inconsistencias.

Cuándo usarlo:
Antes de entregar el modelo o exportar planillas, para asegurarte de que no hay elementos olvidados o duplicados.

Cómo usarlo:

1.       Haz clic en "Planillas vs Modelo"

2.      La herramienta revisa todo y genera un reporte mostrando:

o    Elementos en el modelo que no aparecen en ninguna planilla

o    Filas en planillas sin elemento correspondiente

o    Diferencias entre valores en planillas vs modelo

3.      Revisa el reporte y corrige lo que sea necesario

Ejemplo práctico:
Antes de entregar el modelo al cliente, ejecutas esta herramienta. Te avisa que hay 3 elementos sin planilla asignada. Los revisas, corriges sus categorías y vuelves a verificar hasta que todo esté correcto.

Botón "Revisión Elementos"

Qué hace:
Exporta a Excel toda la información de los elementos del modelo, organizada por planillas, incluyendo una lista de elementos con problemas.

Cuándo usarlo:
Cuando necesitas auditar el modelo completo, revisar datos en Excel o enviar información a otras áreas.

Cómo usarlo:

1.       Haz clic en "Revisión Elementos"

2.      Selecciona la carpeta donde quieres guardar el archivo Excel

3.      La herramienta revisa todos los elementos y genera el Excel

4.      Al terminar, te muestra dónde se guardó el archivo

5.       Abre el Excel y revisa:

o    Una hoja por cada planilla (CM01, CM02, etc.) con todos sus elementos

o    Una hoja de "Excepciones" con elementos que tienen códigos incompletos o incorrectos

Ejemplo práctico:
Antes de coordinar con otras disciplinas, generas este Excel para revisar rápidamente todos los datos del modelo. En la hoja de Excepciones encuentras 5 elementos sin código, los corriges en Revit y vuelves a generar el Excel limpio.

Desplegable "reportes"

Estado: En desarrollo. Este desplegable estará disponible en futuras versiones con herramientas adicionales de generación de reportes avanzados.

Panel 03: "exportacion de Planillas"

Este panel contiene herramientas para generar archivos Excel de planillas de valorización.

Botón "Planillas valorización"

Qué hace:
Crea archivos Excel de planillas de valorización con el formato oficial que requiere el Coordinado Eléctrico Nacional.

Cuándo usarlo:
Cuando necesitas entregar planillas de valorización al Coordinador o a SAESA.

Cómo usarlo:

1.       Haz clic en "Planillas valorización"

2.      Se abre una ventana mostrando todas las planillas disponibles

3.      Puedes:

o    Buscar planillas por nombre usando el cuadro de búsqueda

o    Marcar las planillas que quieres exportar

o    Usar "Seleccionar Todo" para marcar todas

o    Usar "Eliminar Selección" para desmarcar todas

4.      Haz clic en "Exportar Seleccionados"

5.       Espera a que se generen los archivos Excel

6.      Al terminar, aparece un mensaje indicando dónde se guardaron los archivos

7.       Los archivos Excel están listos con el formato correcto (colores, encabezados, estructura)

Ejemplo práctico:
Necesitas enviar las planillas CM01, CM02 y CM03 al Coordinador. Buscas "CM0" en el cuadro de búsqueda, marcas esas tres planillas, exportas y en minutos tienes los tres archivos Excel listos para enviar.

Casos de Uso Prácticos

Caso 1: Iniciar un proyecto nuevo

Situación: Empiezas a trabajar en un proyecto nuevo para el Coordinador Eléctrico Nacional.

Pasos:

1.       Abre el archivo host plantilla de Infotécnica

2.      Ve a menu → proyecto → coordinador

3.      Ingresa todos los datos del proyecto (NUP, nombre, tensión, región, fecha, decreto)

4.      Guarda los datos

5.       Vincula los modelos de las disciplinas como RVT links

6.      Ya puedes empezar a trabajar

Caso 2: Cargar datos iniciales desde Excel

Situación: Ingeniería te envía un Excel con información técnica de 500 elementos.

Pasos:

1.       Verifica que el Excel tenga la columna "inf-id" con los códigos correctos

2.      Ve a menu → datos → Carga de datos

3.      Selecciona el archivo Excel

4.      Espera a que termine la carga

5.       Revisa el resumen: si todos se actualizaron correctamente, listo

6.      Si algunos fallaron, verifica que los códigos coincidan

Caso 3: Cambiar un parámetro en muchos elementos

Situación: Necesitas cambiar el "Estado" de 50 líneas de "Proyectado" a "Existente".

Pasos:

1.       Ve a menu → editar → Parametros por planilla

2.      Selecciona la planilla "CM01 - Líneas de Transmisión"

3.      En la tabla, puedes filtrar por Estado = "Proyectado"

4.      Selecciona las 50 filas que aparecen

5.       Cambia el valor en la columna "Estado" a "Existente"

6.      Guarda los cambios

7.       Los 50 elementos se actualizan instantáneamente

Caso 4: Revisar visualmente elementos por código

Situación: Quieres verificar que todos los elementos con código CM03-00045 están bien ubicados.

Pasos:

1.       Ve a revision → filtrado y seleccion → Por Codigo Interno

2.      Selecciona el código "CM03-00045"

3.      La vista muestra solo esos elementos

4.      Revísalos visualmente

5.       Cuando termines, ve a revision → filtrado y seleccion → Reestablecer vista

6.      Vuelves a ver todo el modelo

Caso 5: Auditar el modelo antes de entregar

Situación: Antes de enviar el modelo al cliente, quieres asegurarte de que todo está correcto.

Pasos:

1.       Ve a revision → herramientas → Planillas vs Modelo

2.      Revisa el reporte de inconsistencias

3.      Corrige los problemas detectados (elementos sin planilla, códigos mal formados)

4.      Ve a revision → herramientas → Revisión Elementos

5.       Genera el Excel de auditoría

6.      Revisa la hoja de "Excepciones"

7.       Corrige los elementos problemáticos

8.      Vuelve a ejecutar ambas herramientas hasta que no haya errores

Caso 6: Exportar planillas de valorización

Situación: Necesitas enviar planillas CM01, CM02 y CM03 al Coordinador.

Pasos:

1.       Ve a exportacion de Planillas → Planillas valorización

2.      En el cuadro de búsqueda, escribe "CM0"

3.      Marca las planillas CM01, CM02 y CM03

4.      Haz clic en "Exportar Seleccionados"

5.       Espera a que se generen los archivos

6.      Ve a la carpeta indicada en el mensaje final

7.       Los archivos Excel están listos con el formato correcto para enviar

Glosario para Usuarios

Términos que debes conocer:

·         CodIntBIM: Código que identifica cada elemento del modelo (ejemplo: CM01-00125)

·         Planilla: Tabla dentro de Revit que organiza elementos por categoría (CM01, CM02, etc.)

·         Parámetro: Información de un elemento (como "Propietario", "Estado", "Tensión")

·         inf-id: Identificador único de cada elemento, generalmente igual al CodIntBIM

·         Excel: Archivo de tabla que usamos para intercambiar información con otras áreas

·         Coordinador Eléctrico Nacional (CEN): Organismo que regula el sistema eléctrico en Chile

·         ViewSchedule: Nombre técnico de las planillas de Revit

·         RVT Link: Modelo de otra disciplina vinculado al archivo principal

·         Exportar: Guardar información del modelo en un archivo externo (Excel)

·         Importar: Traer información desde un archivo externo al modelo

Recomendaciones Finales

Antes de usar las herramientas

·         Lee el texto que aparece al pasar el mouse sobre cada botón

·         Si no estás seguro de qué hace un botón, pregunta a tu coordinador BIM

·         Guarda siempre el modelo antes de usar herramientas que cambian muchos elementos

Durante el trabajo

·         Si aparece un mensaje de error, léelo y cópialo antes de cerrarlo (puede ser útil para soporte)

·         No cierres Revit mientras una herramienta está procesando (espera a que termine)

·         Revisa los resúmenes que muestran las herramientas al finalizar

Si algo sale mal

·         Usa Ctrl+Z para deshacer cambios recientes

·         Si una herramienta no funciona como esperas, contacta al área de soporte BIM

·         Envía capturas de pantalla o mensajes de error completos para que puedan ayudarte mejor
